Features
========

* Reliable and Secure ELK stack
* Ingest large number of log types with minimal configuration
* Distributed Tracing
* Alerting to PagerDuty, Slack, Hipchat, Email etc.
* Minimum 90-day retention with any pricing plan
* 7-day Kafka retention for re-processing or exporting
* Multi-user and Google Auth Kibana Dashboards

Beta Features
-------------

* Scheduled S3 backup to your buckets
* Export logs to HDFS
* Consume logs via Kafka Stream
